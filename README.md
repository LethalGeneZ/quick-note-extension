# Quick Note

This is a simple browser extension I built to jot down thoughts quickly without having to leave the tab. It is designed to be lightweight and stay out of the way.

### What it does
* **Saves automatically:** Anything you type is stored in the browser. You can close the popup or the whole browser, and your note will still be there when you come back.
* **Save to your computer:** If you need to move your note into a document or a folder, you can download it as a regular **.txt file**.
* **Privacy:** I built this to be private. Your notes stay on your device and are never sent to a server or stored in the cloud.

### How to install it
1. **Download the files:** Go to the **Releases** section on the right side of this page and download the latest zip file.
2. **Unzip the folder:** Extract the files to a folder on your computer (put it somewhere permanent, like your Documents folder).
3. **Open your browser settings:** Type `chrome://extensions` (or `brave://extensions`) into your address bar.
4. **Turn on Developer Mode:** Switch the toggle in the top-right corner to **On**.
5. **Load the extension:** Click the **Load unpacked** button and select the folder you just unzipped.

### Why I made this
I wanted a way to bridge the gap between "temporary browser notes" and "actual files on my computer." It uses standard web languages (HTML and JavaScript) and follows modern browser security standards.
